🎓 ProKurs - Modern Kurs ve Öğrenci Yönetim Sistemi

ProKurs, Python ve **CustomTkinter** kullanılarak geliştirilmiş, modern bir kullanıcı arayüzüne sahip (Dark Mode) masaüstü otomasyon programıdır. Eğitim kurumlarının öğrenci, eğitmen ve kurs süreçlerini yönetmesi için tasarlanmıştır. Veritabanı olarak **SQLite** kullanılmıştır.

## 🚀 Özellikler

* **📊 Gelişmiş Dashboard:** Anlık eğitmen, öğrenci, kurs sayıları ve otomatik hesaplanan beklenen gelir (ciro) istatistikleri.
* **👨‍🏫 Eğitmen Yönetimi:** Sisteme uzmanlık alanlarıyla birlikte yeni eğitmen ekleme ve silme.
* **🎓 Öğrenci Yönetimi:** Öğrenci kayıtları oluşturma ve canlı arama (Live Search) ile kolayca bulma.
* **📚 Kurs Yönetimi:** Kontenjan ve fiyat belirleyerek kurs açma, kurs detaylarını (çift tıklayarak) görüntüleme.
* **⚡ Hızlı Kayıt:** Sistemdeki öğrencileri seçili kurslara tek tıkla entegre etme.
* **🗄️ Veritabanı Gezgini:** Arka plandaki ham SQLite tablolarını doğrudan uygulama içinden görüntüleme.

## 🛠️ Kullanılan Teknolojiler

* **Dil:** Python 3.x
* **Arayüz (GUI):** CustomTkinter, Tkinter (ttk)
* **Veritabanı:** SQLite3
